### Full Description

RESQ helps you clean up rough text and turn it into readable Markdown.

Paste messy notes, OCR output, copied documentation, or semi-structured text into RESQ and convert it into a cleaner, more usable Markdown draft. The app combines a rules-based parser with structure rescue heuristics so titles, sections, task lists, metadata, quotes, and command blocks are easier to recover.

Use RESQ when you need to:

- clean up OCR output
- rescue raw meeting notes
- turn rough drafts into readable Markdown
- format command snippets and technical notes
- export structured text as Markdown, HTML, or PDF

RESQ is local-first by default, and Ollama enhancement is optional when you want an additional AI polish pass.
